vagrant_centos_defaults
=======================

Run this role on vagrant centos/7 instances, to get the following benefits:
  * colored root prompt
  * htop, nano, wget installed

Example Playbook
----------------

    - hosts: vagrant_centos_defaults
      roles:
      - andrelohmann.vagrant_centos_defaults
